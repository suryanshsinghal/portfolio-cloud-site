# Static Portfolio Website on AWS (S3 + CloudFront)

## Overview
This project demonstrates deployment of a static website using AWS services.

## Tech Stack
- HTML, CSS
- AWS S3 (Storage)
- AWS CloudFront (CDN)

## Features
- Static website hosting
- Global content delivery using CDN
- Public access configuration
